Отправка сообщения

Космическая форма с полем Имя, большим текстовым блоком до 9000 символов и отправкой прямо тебе на почту.

Максимум 9000 символов 0 / 9000

Что нужно сделать один раз

  1. Создай форму на Formspree.
  2. Скопируй свой адрес вида https://formspree.io/f/xxxxx.
  3. Вставь его в файл docs/index.html вместо PASTE_YOUR_FORMSPREE_ID.
  4. Загрузи папку docs в GitHub и включи GitHub Pages.
